1. Lead Source Field — Add New Values

Go to LeadSquared → Settings → Lead Fields → Lead Source (dropdown) → Add the following values. This field already exists — just add the new values below.

Value to Add Where it comes from Action
Meta Leads from Meta (Facebook/Instagram) paid ads clicking through to website Add New
Instagram_DM Leads from Instagram DM links sent manually Add New
Referral Leads referred by someone, clicking a referral link Add New
Webinar Leads who registered via the Webinar registration form on the website Add New
Website Lead Form Default — leads from /get-started page with no UTM source (organic/direct) Verify Exists
Important: The value in the Source dropdown must match the utm_source parameter in the URL exactly (case-sensitive). E.g. if the ad link uses utm_source=Meta, the dropdown value must be Meta not meta.

4. Two Website Forms — What Each Sends

Both forms push data to LeadSquared via the same API endpoint

Get Started Form — /get-started

Full 4-step lead capture form. Sends all fields.


Fields sent: Name, Email, Phone, Source (from UTM), Degree, Destinations, Start Year, Course Interests, Academic Score, Investment Budget, UTM Campaign, UTM Medium, UTM Adset, UTM Ad

Webinar Registration Form — /webinar

Short registration form. Sends minimal fields.


Fields sent: Name, Email, Phone, Source = Webinar (hardcoded)


All other fields will be blank for webinar leads.

How the Full Flow Works

End-to-end from ad click to LeadSquared CRM entry

User sees ad / DM / referral link
→
Clicks link with UTM tags
→
Lands on /get-started
→
Fills 4-step form
→
Submits
→
Saved to website DB
→
Lead appears in LeadSquared with full UTM data
The website reads UTM parameters from the URL automatically when the page loads. The user never sees or interacts with UTMs — they just fill the form normally. UTM data is captured silently in the background.

UTM Parameter Structure

5 parameters used — each maps to a field in LeadSquared

UTM Parameter LeadSquared Field Purpose Example Value
utm_source Source Which platform the lead came from Meta, Instagram_DM, Referral
utm_medium mx_UTM_Medium Type of traffic paid_social, social_dm, referral
utm_campaign mx_UTM_Campaign Campaign name Masters_India_April2026
utm_adset mx_UTM_Adset Ad set / audience name India_18-25_Lookalike
utm_ad mx_UTM_Ad Specific ad creative name VideoAd_Testimonial_V2

UTM Setup Per Source

How to build links for each channel

Meta Paid Ads (Facebook / Instagram)

In Meta Ads Manager → Ad level → URL Parameters, add the following. Meta will auto-fill campaign/adset/ad names dynamically for every ad.

https://onepercent.in/get-started?utm_source=Meta&utm_medium=paid_social&utm_campaign={{campaign.name}}&utm_adset={{adset.name}}&utm_ad={{ad.name}}

The {{campaign.name}} etc. are Meta dynamic parameters — Ads Manager replaces them automatically.

Instagram / WhatsApp DM Links

Manually created link — paste into DMs. Can add a campaign name to track which DM campaign it was.

https://onepercent.in/get-started?utm_source=Instagram_DM&utm_medium=social_dm&utm_campaign=April_Outreach

Referral Links

Links shared by existing students or partners. No referral code needed at this stage.

https://onepercent.in/get-started?utm_source=Referral&utm_medium=referral

Webinar Registrations

Source is hardcoded in the website — no UTM needed on the link. All webinar leads automatically appear as Source = Webinar in LeadSquared.

https://onepercent.in/webinar ← no UTM needed, source is set automatically

Organic / Direct Traffic

No UTM on the link. LeadSquared will show Source = "Website Lead Form" (default fallback).

https://onepercent.in/get-started ← no UTM, defaults to "Website Lead Form"

Implementation Checklist

Steps for the LeadSquared team in order

1

Add Source dropdown values

Settings → Lead Fields → Source → Add: Meta, Instagram_DM, Referral, Webinar. Verify Website Lead Form already exists.

2

Create all 10 custom fields

Settings → Lead Fields → Add Field for each: Degree, Destinations, Start Year, Course Interests, Academic Score, Investment Budget, UTM Campaign, UTM Medium, UTM Adset, UTM Ad.

3

Verify standard fields are active

Confirm FirstName, EmailAddress, Phone, Source fields are active and visible in lead detail view.

4

Test with a live submission

Submit a test lead on onepercent.in/get-started and verify it appears in LeadSquared with all fields populated correctly.

5

Test UTM tracking

Submit a test lead using a UTM link like:
onepercent.in/get-started?utm_source=Meta&utm_campaign=Test
Verify Source = Meta and UTM Campaign = Test in LeadSquared.

6

Set up Meta Ads UTM parameters

In Meta Ads Manager, add URL parameters to all active ads using the dynamic template from the UTM Plan tab.
